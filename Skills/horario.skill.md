# Reglas de Horarios
- Siempre usar formato 12h (a.m./p.m.).
- Las actualizaciones de horarios de "Rutina del Día" deben reflejarse tanto en componentes de texto como en metadatos de SEO si existen.
- Al finalizar, verificar con `npm run test` (si aplica) y mostrar captura en el navegador.
